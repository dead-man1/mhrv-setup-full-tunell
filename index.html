⚙️ اطلاعات مشترک — یه بار وارد کن

این ۳ مقدار بین کد VPS، کد Apps Script و اپ اندروید مشترک هستن. وقتی اینجا وارد کنی، کد همه سربرگ‌ها خودکار درست میشه — فقط کپی کن.

آدرس IP سرور VPS
IP عمومی VPS هلند — بعد از نصب با دستور curl -4s ifconfig.me بدستت میاد
پورت Tunnel Node (پیش‌فرض ۸۰۸۰)
پورتی که tunnel-node روش listen می‌کنه — معمولاً ۸۰۸۰ رو دست نزن
پورت SSH سرور
پورتی که با Termius بهش وصل میشی — مهمه که اشتباه نباشه
رمز مشترک (AUTH KEY)
رمز دلخواه — حداقل ۱۵ کاراکتر، ترکیب حرف و عدد. این رمز در VPS، Apps Script و اپ باید یکی باشه
⚠️ اطلاعات بالا رو وارد کن تا کدها آماده بشن

📋 خلاصه — بعد از پر کردن

۱. برو سربرگ کد VPS → کد رو کپی کن → توی Termius اجرا کن

۲. برو سربرگ Apps Script → کد رو کپی کن → deploy کن → ID بگیر

۳. برو سربرگ اندروید → اپ رو نصب کن → ID رو وارد کن → Connect

🖥️ این کد رو در Termius روی VPS اجرا کن. اطلاعات از سربرگ «تنظیمات» خودکار وارد شده.

🚀 مراحل Deploy

    
۱. script.google.com → New project → کد پیش‌فرض رو پاک کن

    
۲. کد بالا رو Paste کن → آخر فایل چک کن خط اضافه نباشه

    
۳. Deploy → New deployment → Web app

    
Execute as: Me | Who has access: Anyone

    
۴. هشدار «unsafe»: Advanced → Go to (unsafe) → Allow

    
۵. Deployment ID رو کپی کن — فقط ID، نه کل URL

📋 تنظیم اپ

    
۱. APK رو نصب کن (نصب از منابع ناشناس باید فعال باشه)

    
۲. Deployment ID رو وارد کن — فقط ID، نه کل URL

    
۳. Auth key: رمز وارد نشده

    
۴. Google IP: 216.239.38.120 یا دکمه Auto-detect

    
۵. Front domain: www.google.com — دست نزن

    
۶. دکمه Install MITM Certificate رو بزن

    
۷. بخش SNI pool → تست بزن → ۲-۳ آدرس با کمترین پینگ انتخاب کن

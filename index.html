🧹 پاک‌سازی (اختیاری)

cleanup.sh
docker rm -f tunnel-node 2>/dev/null || true
docker rmi ghcr.io/therealaleph/mhrv-tunnel-node:latest 2>/dev/null || true
ufw --force reset && ufw --force disable
echo "✅ پاک شد"

  • ⚠️از جیمیل شخصی اصلی استفاده نکن

🔧 عیب‌یابی

  • 🔴
    Relay timeout
    Apps Script درست deploy نشده — دوباره deploy کن
  • 🔴
    Syntax Error deploy
    خط آخر اضافه رو پاک کن
  • 🔴
    VPS قفل شد
    پورت SSH رو قبل از ufw enable باز کن. اگه قفل شد → OS reinstall
  • 🔴
    Docker نصب نشد (dpkg error)
    dpkg --configure -a بزن، بعد دوباره install کن
  • 🔴
    اتصال قطع و وصل
    deployment معیوب خودکار ۲ دق blacklist میشه — طبیعیه (v1.8.5)
